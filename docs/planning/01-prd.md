# PRD (Product Requirements Document)

## 프로젝트 개요

### 서비스명
**Space Moduler** (가제)

### 한 줄 설명
2D 평면도 이미지를 드래그 앤 드롭 하나로 3D 렌더링해주는 웹 서비스

### 태그
#Web3D #InteriorTech #AI #VisionAPI

---

## 비전

### 핵심 가치
> **"평면도 하나로, 내 집을 3D로"**

사용자가 복잡한 절차 없이 평면도 이미지만 올리면 즉시 3D로 확인할 수 있게 한다.

### 롱텀 비전 (1~3년)
- 단순 평면도 → 3D를 넘어, 실제 가구 배치 시뮬레이션
- AR/VR 연동으로 실제 공간에서 가구 배치 확인
- 인테리어 쇼핑 연동 (3D에서 가구 바로 구매)

---

## 문제 정의

### 사용자가 겪는 문제

| 문제 | 설명 | 사용자 언급 |
|------|------|-----------|
| **단계 복잡** | 기존 서비스는 회원가입, 업로드, 설정, 수정 등 여러 단계 필요 | "단계가 너무 많아요" |
| **UI 불편** | 직관적이지 않은 인터페이스, 사용법 학습 필요 | "UI가 불편해요" |
| **속도 느림** | 3D 변환에 너무 오래 걸림 | "속도가 너무 느려요" |

### 경쟁사 분석

| 서비스 | 장점 | 단점 |
|--------|------|------|
| Homestyler | 기능이 많음 | 한글 미지원, 복잡함 |
| Planner 5D | 디자인 좋음 | 유료, 속도 느림 |
| SketchUp | 전문적임 | 학습 곡선 가파름 |
| Coohom | 한국형 | UI 복잡, 유료 |

### 우리의 해결책
- **드래그 앤 드롭 1회** - 이미지 올리면 끝
- **간단한 UI** - 클릭 최소화
- **AI 자동 인식** - Gemini 3.0 Pro Vision API로 벽/문/창문 자동 인식
- **빠른 처리** - 10~30초 내 완료 (로딩 UX로 커버)

## 타겟 사용자

### 1차 타겟 (Primary)
- **인구통계**: 20~40대, 자가/전세 세입자, 이사/인테리어 계획 중
- **니즈**: 평면도로 공간 감각을 빠르게 파악하고 싶음
- **행동**: 온라인으로 아파트/빌라 평면도 검색

### 2차 타겟 (Secondary)
- 인테리어 디자이너 (고객 보여주기용)
- 부동산 중개업자 (매물 홍보용)

### 페르소나

#### 페르소나 1: "수진" (28세, 서울 거주)
- **상황**: 강남구 전세 25평 아파트로 이사 예정
- **목표**: 이사 전에 가구 배치 미리 시뮬레이션
- **고통점**: 평면도만으로는 공간 감각이 안 잡힘
- **우리가 해결**: 평면도 올리면 3D로 바로 확인

#### 페르소나 2: "현우" (35세, 인테리어 업체)
- **상황**: 고객에게 시뮬레이션 보여줘야 함
- **목표**: 빠르고 간단하게 3D 렌더링
- **고통점**: 기존 툴은 복잡하고 시간 오래 걸림
- **우리가 해결**: 1분 안에 결과 확인 가능

## 비기능 요구사항

### 성능
- **업로드**: 3초 이내
- **AI 인식**: 10~30초 (로딩 UX로 커버)
- **3D 렌더링**: 프레임레이트 30fps 이상

### 호환성
- **브라우저**: Chrome, Edge, Safari (최신 2버전)
- **디바이스**: 데스크톱 우선, 태블릿 지원
- **이미지**: JPG, PNG, PDF (최대 10MB)

### 접근성
- **언어**: 한국어 우선
- **반응형**: 1920x1080, 1366x768 지원
